Creative Scala

Creative Scala's goal is to make Scala fun. We do this by creating training material that introduces Scala through non-traditional means, and libraries that unleash digital creativity.

Learning

Creative Scala: Form and Function is an introduction to programming and Scala, aimed at students without a STEM background. The second edition is currently in development.

We also have several extended case studies that expand on and go further than the material in Creative Scala:

  • Machine learning by gradient descent
  • Forms
  • Parser combinators

Code

  • Doodle is a library for compositional two dimensional vector graphics. It has applications for data visualization and creative coding.
  • Doodle SVG adds SVG support to Doodle.
  • Doodle Explore adds interactive visualizations to Doodle.
